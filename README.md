# cs-data-modeling-produfol
Projeto de Banco de Dados SQL para a Produfol (Comércio de Fertilizantes). Contém scripts DDL/DML para gestão de estoque por lotes, vendas e clientes. Modelagem normalizada (3FN) com scripts de criação, inserção, consultas complexas, etc. Desenvolvido para a disciplina de Modelagem de Dados, parte do curso de Ciência da Computação.
